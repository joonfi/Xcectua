# Governance Performance

## Purpose
Measure governance cost to ensure it is
**understood**, **predictable**, and **sustainable**.

---

## What Is Measured

- Validation duration
- Variance by change type
- Trend over time

---

## What Is Not Measured

- Code quality
- Policy correctness
- Developer performance

---

## Interpretation

Performance data is advisory.
It informs optimization discussions,
not enforcement decisions.
